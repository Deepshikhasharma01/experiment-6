# experiment-6
**Exp 2.1 – Middleware:**
Middleware is a backend layer that processes client requests before reaching the server. It is used for tasks like authentication, logging, validation, and error handling, improving security and code structure.

**Exp 2.2 – JWT (Banking API):**
JWT is used for secure authentication. After login, a token is generated and sent to the client, which is then verified for accessing protected routes, ensuring only authorized users can use banking services.

**Exp 2.3 – Transaction System:**
A transaction system handles banking operations like deposit, withdrawal, and money transfer. It verifies users, checks balances, and updates the database while maintaining security and data accuracy.
